=== Custom Card Carousel Pro ===
Version: 2.0.0
Requires at least: 5.8
Tested up to: 6.5
Requires PHP: 7.4
License: GPLv2

Full style-control card carousel for Elementor.

== Installation ==
1. Upload zip via Plugins → Add New → Upload Plugin
2. Activate
3. Go to Appearance → Card Carousel to add/edit slides
4. Drag "Card Carousel" widget from Elementor General panel
5. Use the STYLE tab for full visual control

== Elementor Style Controls ==
- Cards per view (desktop / tablet / mobile independently)
- Card background, border, radius, shadow, padding
- Image aspect ratio
- Badge colours, radius, font size
- Title typography (font, size, weight, line-height, colour)
- Description typography and colour
- Button: normal + hover states, typography, padding, radius, full-width toggle
- Arrow: size, background, icon colour, border, radius (normal + hover)
- Pagination dots: colour, active colour, size, active width
- Carousel spacing (top/bottom padding)
