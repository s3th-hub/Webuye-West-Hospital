# Webuye West Hospital Website

A modern, responsive hospital website built using WordPress and Elementor, designed to provide clear access to healthcare services and improve user navigation.

## 🔗 Overview
This project focuses on delivering a clean and user-friendly interface for a hospital environment, allowing patients and visitors to easily explore services, access information, and navigate the website efficiently.

## 🚀 Features
- Responsive design (mobile, tablet, desktop)
- Structured clinical services pages
- Sidebar navigation for easy access to different services
- Clean and modern healthcare UI
- Fast and optimized performance
- Easy content management using WordPress

## 🛠️ Tech Stack
- WordPress
- Elementor (Hello Elementor Theme)
- HTML5
- CSS3
- JavaScript

## 🧩 My Role
- Designed and developed the website layout
- Structured service pages for better user experience
- Implemented responsive design across all devices
- Customized WordPress components and layouts
- Ensured usability and accessibility of hospital services
